# Predicate for iRODS Connectivity

A predicate to check whether you are currently connected to an iRODS
server.

## Usage

``` r
is_connected_irods(...)
```

## Arguments

- ...:

  Currently not implemented.

## Value

Boolean whether or not a connection to iRODS exists.

## Examples

``` r
is_connected_irods()
#> [1] FALSE
```
