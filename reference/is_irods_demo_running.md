# Predicate for iRODS Demonstration Service State

A predicate to check whether you are running iRODS docker demo
containers.

## Usage

``` r
is_irods_demo_running(...)
```

## Arguments

- ...:

  Currently not implemented.

## Value

Boolean whether or not connected to iRODS

## Examples

``` r
is_irods_demo_running()
#> [1] FALSE
```
